CMS's SSA to FIPS CBSA and MSA County Crosswalk

CMS periodically produces SSA to FIPS CBSA to county crosswalk files. They released a CBSA to MSA to FIPS county crosswalk as well. Some CMS data files have SSA state and county codes or county name rather than FIPS state and county codes. Jean Roth processed the data files below for greater ease of use.

Updates and changes.


The MSA CBSA FIPS County crosswalk below is a merge of the individual CBSA and MSA files below.

Users should be aware that it appears that CMS did not update their SSA to FIPS county crosswalk to use 2010 Census definitions. It should be possible; however, to link the FIPS county codes in this file to those in Census' CBSA to FIPS County Crosswalk file to get more up to date CBSA definititions.

Some additional notes about the data are the following:

Los Angeles FIPS 06037 can have two SSA county codes: 05210 and 05200.
Los Angeles is the only FIPS county with multiple SSA codes.
2012-2016 files are identical.
2005 and 2011 file have 3,289 counties and 2012-2016 have 3,273 counties.
A list of the counties present in 2011 and not in 2012 is in Updates and changes
